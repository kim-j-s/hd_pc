다른사람의 보험 가입 절차를
확인해주세요

피보험자가 배우자/부모/자녀인 경우 아래의 순서로 가입이 진행됩니다.

  1. 01
    고객정보입력

    계약자(보험료를 납입하실 분)와 피보험자의 기본정보를 입력합니다.

  2. 02
    피보험자 동의 및 보험료 확인

    피보험자에게 발송되는 "피보험자 개인(신용)정보 수집이동동의 안내" 알림톡에서 확인 가능합니다. (당일 24시까지 확인 가능)

  3. 03
    계약자 가입 내용 확인

    계약자에게 발송되는 "피보험자 개인(신용)정보 수집이용동의 완료 안내" 알림톡에서 확인 가능합니다. (당일 24시까지 확인 가능)

  4. 04
    보험료 결제 및 가입완료

    가입완료 후 피보험자에게 발송되는 "보험가입 완료 안내" 알림톡에서 가입 내용 확인이 가능합니다.

  • 연말정산 소득공제는 계약자 기준으로 제공되며, 피보험자가 소득이 있는 경우에는 소득공제 혜택이 적용되지 않습니다.
  • 보험료 계산시 사망보장을 선택하신 경우, 관련 법률에 따라 사망 보장은 자동으로 제외됩니다.
