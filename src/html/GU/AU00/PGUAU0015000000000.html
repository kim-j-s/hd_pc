PC에 저장된 공동인증서를 스마트폰으로 복사할 수 있어요

{{> cp/button type="1" size="2" b-text="공동인증서 내보내기"}}

이용절차

  1. 스마트폰 인증센터 메뉴에서 [공동인증서 복사(PC→스마트폰)] 선택하여 인증번호 발급
  2. PC 인증센터 메뉴에서 [공동인증서 복사 (PC→스마트폰)]을 선택
  3. PC 화면에서 복사할 공동인증서 선택 후 비밀번호 입력 후 [다음] 선택
  4. PC 화면에서 스마트폰을 통해 발급받은 12자리 인증번호 입력 후 [마침] 버튼 선택
  5. 스마트폰 화면에서 [공동인증서 가져오기] 누르면 복사 완료
{{#> cp/notice dp-active=true }}
  • PC에서 복사한 공동인증서가 스마트폰에서 바로 확인되지 않을 경우, 현대해상 앱을 삭제한 후 다시 설치해주세요.
