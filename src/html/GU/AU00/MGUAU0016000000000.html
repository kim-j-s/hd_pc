에서
PC
으로 복사하기

이용절차

  1. 스마트폰에서 PC로 복사할 인증서 선택
  2. 인증서 비밀번호 입력 후
    [인증서 내보내기]를 위한 인증번호 확인
  3. 인증센터 메뉴에서
    [공동인증서 복사(스마트폰 → PC)] 클릭
  4. 스마트폰에서 발급받은 인증번호 입력
  5. 스마트폰에서 [공동인증서 내보내기]
    PC에서 [공동인증서 가져오기]
