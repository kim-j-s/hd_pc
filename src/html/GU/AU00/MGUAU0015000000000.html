이용절차

  1. 인증번호 발급
  2. 인증센터 메뉴에서 공동인증서 복사(PC→스마트폰)을 선택
  3. 복사할 공동인증서 선택 후 비밀번호 입력
  4. PC화면에서 인증번호 12자리를 입력
